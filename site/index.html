האתר עלה בהצלחה

מטלת סיכום DevOps

אתר זה הוקם אוטומטית באמצעות Jenkins Pipeline, Terraform ו-Ansible. התשתית נוצרה בענן, השרת הוגדר, ו-Nginx מגיש את הדף הזה.

GitHub שמירת קוד הפרויקט, התיעוד וקבצי ההגדרה.
Terraform הקמת שרת, Security Group ומשאבים נדרשים בענן.
Ansible התקנת Nginx והעתקת קבצי האתר לשרת.
Jenkins הרצת התהליך המלא ובדיקת זמינות האתר.
DevOps final project - automated infrastructure and deployment.
